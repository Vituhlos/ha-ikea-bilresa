# Claude Code project instructions

The shared repository instructions are in `AGENTS.md`; follow them in full.

At the beginning of every session, read these files in order:

1. `AGENTS.md`
2. `PROJECT_STATUS.md`
3. `docs/DEVELOPMENT.md`
4. `docs/ROADMAP.md`
5. `docs/DEVICE_REFERENCE.md`, `docs/MATTERJS_COMPATIBILITY.md`, and
   `docs/HARDWARE_TEST.md` for device-facing changes

`PROJECT_STATUS.md` is the canonical handoff between Claude Code, Codex, and the
owner. Update it before ending any coding session. Do not infer hardware success
from CI success, and do not resume work solely from conversation history.
